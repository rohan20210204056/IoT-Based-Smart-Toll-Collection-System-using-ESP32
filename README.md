# 🚗 IoT-Based Smart Toll Collection System using ESP32  

---

## 📌 Project Overview  

### 🚨 Problem  
Traditional toll collection systems suffer from several issues:  

1. Traffic congestion due to manual toll collection  
2. Time-consuming payment process  
3. Fuel wastage from long queues  
4. Human errors in transactions  
5. Lack of real-time monitoring  
6. High operational costs  

---

## 💡 Proposed Solution  

The IoT-Based Smart Toll Collection System automates toll operations using RFID, ESP32, and cloud integration.

### Key Features:

- Automatic vehicle identification using RFID  
- Real-time toll deduction via cloud  
- Automated gate control using servo motor  
- Data logging in Google Sheets  
- Web dashboard using HTML & CSS  
- Buzzer alerts for system feedback  

---

## 🧩 Key Components  

- ESP32 Microcontroller  
- RFID Reader (RC522)  
- RFID Tags  
- IR / Load Sensor  
- Servo Motor  
- Buzzer  
- Google Apps Script  
- Google Sheets  
- Web Interface (HTML/CSS)

## ⚙️ Operation  

1. Vehicle arrives at toll  
2. Sensor detects vehicle  
3. RFID tag is scanned  
4. Data sent to cloud  
5. System verifies balance  
6. Gate opens if valid, else denied  
7. Buzzer gives feedback  
8. Transaction stored in database
